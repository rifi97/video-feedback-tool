# Video Feedback Tool - 업데이트 노트

## v1.2.1 (2026-05-12)

### 🔧 v1.2.1 변경사항
- **수정**: 내보내기 후 Notion에 붙여넣을 때 일반 텍스트가 아니라 체크박스 목록으로 인식되도록 개선
  - 클립보드에 Notion이 해석할 수 있는 HTML 체크리스트 형식을 함께 저장
  - 기존 `- [ ] 11:12 피드백` 일반 텍스트 형식은 호환용으로 유지

## v1.2.0 (2026-05-12)

### ✨ 주요 기능
- **노션 피드백 불러오기**: 클립보드에 복사된 노션 To-do 피드백을 앱 피드백 목록으로 가져오기
  - `- [ ] 07:38 피드백 내용` 형식 지원
  - 불러온 피드백은 기존 피드백처럼 클릭 시 해당 타임스탬프로 이동

## v1.0.7 (2024-01-30)

### 🎬 주요 기능
- **동영상 재생**: 로컬 MP4, MOV, M4V 파일 재생 지원
- **타임스탬프 피드백**: 현재 재생 시간과 함께 피드백 기록
- **노션 내보내기**: To-do 리스트 마크다운 형식으로 클립보드 복사
- **드래그 앤 드롭**: 비디오 파일을 앱에 드래그하여 바로 열기

### ⌨️ 단축키
| 단축키 | 기능 |
|--------|------|
| `Space` | 재생/일시정지 |
| `←` | 한 프레임 뒤로 |
| `→` | 한 프레임 앞으로 |
| `⌘E` | 피드백을 클립보드로 내보내기 |

### 🔧 v1.0.7 변경사항
